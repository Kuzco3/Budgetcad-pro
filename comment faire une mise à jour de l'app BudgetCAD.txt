comment faire une mise à jour de l'app BudgetCAD : 

Étape 1 : Mettre à jour GitHub
bash
# Dans le dossier de votre projet local
git add .
git commit -m "Mise à jour de l'application"
git push
Étape 2 : Régénérer l'APK
Dans Node.js command prompt :

bash
cd C:\Users\maham
rmdir /s app
del android.keystore
bubblewrap init --manifest=https://kuzco3.github.io/Budgetcad-pro/manifest.json --icon-url=https://kuzco3.github.io/Budgetcad-pro/icon-512.svg
Puis répondez aux questions (mêmes réponses que la première fois) :

Password : budgetcad (le même)

Puis :

bash
bubblewrap build
Étape 3 : Installer la nouvelle version
Transférez le nouveau app-release-signed.apk sur votre téléphone

Ouvrez-le

Il vous dira "Voulez-vous mettre à jour cette application ?"

Cliquez Oui

⚠️ Important : Gardez le même keystore
Si vous voulez faire des mises à jour sans désinstaller, il faut garder le même fichier android.keystore. Ne le supprimez pas. Utilisez les mêmes mots de passe.

Procédure sans suppression du keystore :

bash
cd C:\Users\maham
# Supprimez SEULEMENT le dossier app, PAS le keystore
rmdir /s app
# Recréez le projet (le keystore existant sera réutilisé)
bubblewrap init --manifest=https://kuzco3.github.io/Budgetcad-pro/manifest.json --icon-url=https://kuzco3.github.io/Budgetcad-pro/icon-512.svg
# Construisez
bubblewrap build
