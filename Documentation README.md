# BudgetCAD Pro v5.0

![Version](https://img.shields.io/badge/version-5.0.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)

**Gestionnaire financier personnel expert avec Intelligence Artificielle**

## 🚀 Fonctionnalités

### 💰 Gestion financière
- Dashboard temps réel avec KPIs
- Journal des dépenses avec filtres avancés
- Calendrier interactif
- Scanner de factures OCR (Tesseract.js)
- Budgets personnalisables avec alertes

### 🤖 Intelligence Artificielle
- Prédictions de dépenses (TensorFlow.js)
- Détection d'anomalies statistiques
- Score de santé financière
- Recommandations personnalisées
- Assistant IA conversationnel

### 🔐 Sécurité bancaire
- Chiffrement AES-256 des données
- Authentification biométrique (Face ID / Touch ID)
- Journal d'audit complet
- Sauvegardes automatiques
- Effacement d'urgence

### 📊 Visualisations
- Graphiques interactifs (Chart.js)
- Heatmap d'intensité des dépenses
- Analyse multi-dimensions
- Simulation de scénarios

### 📱 PWA
- Installation sur tous les appareils
- Mode hors-ligne complet
- Synchronisation automatique
- Notifications push

## 🛠️ Technologies

- **Frontend**: HTML5, CSS3, JavaScript (ES2024)
- **Stockage**: IndexedDB + localStorage
- **IA**: TensorFlow.js, régression linéaire
- **OCR**: Tesseract.js
- **Cartes**: Leaflet + OpenStreetMap
- **Sécurité**: CryptoJS AES-256, WebAuthn
- **Graphiques**: Chart.js 4.4
