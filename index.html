📚 Guide d'utilisation

Bienvenue dans BudgetCAD ! Ce guide vous explique tout ce que vous devez savoir pour gérer votre budget facilement.

🚀 DÉMARRAGE RAPIDE (Première utilisation)

Étape 1 : Cliquez sur 💵 Revenus dans le menu de gauche.

→ Entrez votre taux horaire (ex: 25,00 $/h) et vos heures par semaine (ex: 35 h).

→ Le calcul du salaire NET se fait automatiquement (environ 70% du brut pour le Québec).

Étape 2 : Ajoutez votre salaire réel quand vous recevez votre paie.

→ Bouton « ＋ Ajouter un salaire réel » : entrez la date et le montant reçu.

→ 💡 Pourquoi faire ? Sans salaire réel, le Dashboard reste à 0 $. Cela évite de fausser vos calculs.

Étape 3 : Retournez au 🏠 Dashboard et commencez à ajouter vos dépenses.

→ Bouton bleu « + Dépense » en haut à droite.

→ Remplissez le lieu, la catégorie et le montant. C'est tout !

🏠 DASHBOARD (Page d'accueil)

C'est votre tableau de bord principal. Il résume toute votre situation financière.

💚 Revenus : L'argent que vous avez gagné ce mois. Se base sur vos salaires réels. Si vous n'avez pas encore reçu de paie ce mois-ci, ce sera 0 $.

💙 Dépenses : Tout ce que vous avez dépensé ce mois. Addition de toutes vos dépenses (sauf Épargne et Non-dépensé).

Solde : Revenus − Dépenses − Non-dépensé. Exemple : vous gagnez 2000 $, dépensez 1500 $ → Solde = +500 $.

💜 Épargne (mois) : Ce que vous avez mis de côté ce mois (dépenses type « Épargne » + Solde positif).

💜 Épargne (année) : Cumul de toute votre épargne depuis janvier. Exemple : 200 $ en janvier + 300 $ en février = 500 $.

🟡 Non-dépensé : Argent restant non utilisé. Calculé automatiquement.

📅 Sélecteur d'année : Changez l'année pour voir vos anciennes données ou planifier l'avenir.

📈 Graphique Dépenses vs Revenus : Visualisez vos 6 derniers mois en un coup d'œil.

🥧 Graphique Catégories : Voyez où va votre argent (épicerie, loyer, loisirs...).

📋 Récapitulatif mensuel : Tableau complet mois par mois avec une ligne TOTAL en bas.

⏱️ Dernières dépenses : Les 6 dépenses les plus récentes.

📝 JOURNAL DES DÉPENSES

Retrouvez toutes vos dépenses au même endroit. Filtrez, modifiez, supprimez.

Filtre Mois : Affiche uniquement les dépenses d'un mois précis.

Filtre Catégorie : Affiche seulement l'épicerie, ou seulement l'essence...

Recherche : Tapez un mot (ex: « IGA », « Shell ») pour trouver rapidement.

Min $ / Max $ : Filtrez par montant. Exemple : Min 50 $, Max 200 $.

✎ Modifier : Passez la souris sur une dépense, cliquez sur le crayon.

✕ Supprimer : Passez la souris sur une dépense, cliquez sur la croix.

📥 CSV : Exportez toutes vos dépenses pour les ouvrir dans Excel.

➕ AJOUTER UNE DÉPENSE (Le formulaire)

Remplissez ce formulaire à chaque achat ou paiement.

📅 Date : Pré-remplie avec aujourd'hui. Modifiez si besoin.

🕐 Heure : Optionnelle. Pratique pour les reçus.

📍 Lieu / Endroit : Nom du magasin ou commerce. L'autocomplétion vous propose les lieux déjà utilisés.

📝 Description : Ce que vous avez acheté. Exemple : « Épicerie semaine », « Plein d'essence ».

🏷️ Catégorie : Choisissez parmi la liste ou tapez une nouvelle. Exemples : Épicerie, Carburant, Loyer, Restaurant...

💲 Montant : En dollars canadiens. Utilisez le point pour les cents. Exemple : 42,99

🏷️ Type de dépense :

• Fixe : Même montant chaque mois (loyer, abonnement, assurance).

• Variable : Change chaque mois (épicerie, essence, resto).

• Occasionnel : Arrive de temps en temps (cadeau, réparation, voyage).

• Épargne : Argent mis de côté (CELI, REER, compte épargne).

• Non-dépensé : Argent qui reste après tout (calculé automatiquement).

💬 Note : Commentaire libre. Exemple : « Payé comptant », « Facture #1234 ».

📅 CALENDRIER

Visualisez vos dépenses sur un calendrier mensuel.

◀ ▶ : Passez au mois précédent ou suivant.

● : Revenir au mois en cours.

Points bleus : Jours avec des dépenses. Le montant total du jour est affiché.

Encadré bleu : Date d'aujourd'hui.

📷 SCANNER DE FACTURES (OCR)

Prenez une photo de votre ticket de caisse, l'application lit les articles automatiquement !

1. Cliquez sur la zone « Photographier ».

2. Prenez une photo claire de votre reçu.

3. L'OCR analyse l'image et extrait les articles avec leurs prix.

4. Cochez/décochez les articles à importer.

5. Cliquez sur « Importer ».

⚠️ Conseil : Vérifiez les montants, l'OCR peut parfois se tromper.

💰 BUDGET (Suivi par catégorie)

Définissez une limite de dépenses pour chaque catégorie et suivez votre progression.

Barre de progression :

• Vert : Vous êtes dans votre budget (moins de 85%).

• Orange : Attention, vous approchez de la limite (85-100%).

• Rouge : Budget dépassé ! (plus de 100%).

⚙️ Modifier les budgets : Cliquez pour entrer vos limites par catégorie. Exemple : Épicerie = 400 $/mois, Restaurant = 150 $/mois.

🚨 Alertes Dashboard : Une alerte rouge apparaît quand vous dépassez un budget.

📈 GRAPHIQUES & ANALYSES

Visualisez vos finances avec de beaux graphiques.

📊 Onglet Dépenses :

• Anneau : Répartition de vos dépenses par catégorie (comme un camembert).

• Progression radiale : Cercles montrant votre consommation par rapport au budget. Si vous dépassez, vous verrez « +150% » en rouge.

• Jauge du mois : Deux jauges circulaires (Dépenses et Épargne) avec le solde au centre.

💵 Onglet Salaires :

• Barres : Vos salaires réels par semaine.

• Cumul vs Estimé : Compare ce que vous avez vraiment gagné avec ce qui était prévu.

📅 Mois : Utilisez le sélecteur en haut pour choisir le mois à analyser.

📊 COMPARAISON (Mois par mois)

Comparez vos dépenses sur toute l'année.

Graphique à barres empilées : Chaque barre = un mois. Chaque couleur = une catégorie. Vous voyez d'un coup d'œil quel mois a été le plus dépensier et dans quelle catégorie.

Tableau : Évolution mois par mois avec la tendance.

• 📈 Hausse : dépenses en augmentation.

• 📉 Baisse : dépenses en diminution.

• ➡️ Stable : dépenses similaires.

💵 REVENUS (Gestion complète)

Tout ce qui concerne l'argent qui entre.

💰 Salaires réels : Enregistrez chaque paie. Date, heure (optionnelle), montant, note. Cliquez sur ✕ pour supprimer.

📊 Statistiques : Salaire moyen, minimum et maximum calculés automatiquement.

📊 Salaire estimé : Taux horaire × Heures/semaine. Le NET se calcule automatiquement (70% du brut). Exemple : 25 $/h × 40 h = 1000 $ brut → 700 $ net/semaine.

📥 Revenus supplémentaires : Ajoutez d'autres sources de revenus.

• Mensuel : Chaque mois (allocations, pension alimentaire).

• Trimestriel : Tous les 3 mois (dividendes, remboursements).

• Annuel : Une fois par an (bonus, retour d'impôts).

💾 Enregistrer : N'oubliez pas de cliquer pour sauvegarder.

🏷️ CATÉGORIES (Personnalisation)

Gérez vos propres catégories de dépenses.

➕ Créer : Tapez un nom et cliquez sur +. Exemple : « Animaux », « Sport », « Cadeaux ».

🗑️ Supprimer : Cliquez sur une catégorie existante, puis confirmez.

⚠️ Attention : Supprimer une catégorie ne supprime pas les dépenses associées.

💲 Budgets : Définissez une limite pour chaque catégorie directement ici.

🔄 SYNCHRONISATION (Sauvegarde et transfert)

Transférez vos données entre appareils ou faites une sauvegarde.

📱 QR Code : Exportez vos données en QR code. Scannez-le avec un autre appareil pour importer.

📋 Copier : Copie les données dans le presse-papier. Collez dans un fichier ou un message.

📥 Télécharger JSON : Sauvegarde complète dans un fichier.

📤 Charger fichier : Restaurez une sauvegarde.

📷 Scanner QR : Utilisez la caméra pour scanner un QR code.

Ou collez : Collez des données JSON et importez-les.

⚙️ PARAMÈTRES

Personnalisez l'apparence de l'application.

🎨 Palette : 10 thèmes de couleurs. « Premium » (Platine, Or, Diamant...) et « Classiques ».

Seuil alerte : À quel pourcentage du budget l'alerte se déclenche (85% par défaut).

Enregistrer : Sauvegardez vos paramètres.

⚠️ Effacer données : Supprime TOUT. Action irréversible.

💡 ASTUCES ET CONSEILS

• Mode plein écran : Bouton ◧ en haut. Cache le menu pour plus d'espace.

• Thème clair/sombre : Bouton 🌙/☀️ en haut. L'application détecte automatiquement votre préférence système.

• Autocomplétion : Les lieux et catégories s'affichent automatiquement quand vous tapez.

• Données 100% privées : Tout est stocké dans votre navigateur. Aucune donnée n'est envoyée sur Internet.

• Sans salaire réel = 0 $ : Tant que vous n'avez pas enregistré votre paie, le Dashboard reste à zéro pour éviter les faux calculs.

• Planification : Changez l'année pour 2027 ou 2028 et simulez votre budget futur.

• Export régulier : Faites une sauvegarde JSON de temps en temps (Synchroniser → Télécharger JSON).

• Mot-clé automatique : Si vous tapez « IGA » dans le lieu, la catégorie « Épicerie » est suggérée automatiquement.

❓ FOIRE AUX QUESTIONS

Q : Pourquoi mes revenus sont à 0 $ ?
R : Vous n'avez pas encore ajouté de salaire réel pour ce mois. Allez dans Revenus → Ajouter un salaire réel.

Q : Comment voir l'année dernière ?
R : Dans le Dashboard, utilisez le sélecteur 📅 Année en haut.

Q : Comment changer un budget ?
R : Allez dans Budget → ⚙️ Modifier les budgets, ou dans Catégories → section Budgets.

Q : L'OCR ne lit pas bien mon ticket.
R : Assurez-vous d'avoir une photo bien droite et bien éclairée. Vérifiez les montants avant d'importer.

Q : Comment transférer mes données sur un nouveau téléphone ?
R : Allez dans Synchroniser → QR Code → générez le code → scannez-le avec l'autre appareil.

Q : Le QR Code est trop volumineux.
R : L'application compresse automatiquement les données. Si c'est encore trop gros, exportez par fichier JSON.

Q : Mes données sont-elles en sécurité ?
R : Oui. Tout est stocké localement dans votre navigateur (localStorage). Rien n'est envoyé sur Internet. BudgetCAD fonctionne même sans connexion.
