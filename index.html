Politique de confidentialité et conditions d'utilisation

🔒 Politique de confidentialité

Dernière mise à jour : 5 mai 2026

1. Collecte des données

L'application BudgetCAD ne collecte aucune donnée personnelle. Toutes les informations financières que vous saisissez sont stockées exclusivement sur votre appareil dans le stockage local de votre navigateur.

2. Utilisation des données

Vos données sont utilisées uniquement pour afficher vos tableaux de bord financiers, calculer vos soldes et générer des graphiques. Aucune donnée n'est transmise à un serveur externe.

3. Partage des données

BudgetCAD ne partage, ne vend et ne transmet aucune donnée à des tiers. La synchronisation se fait via QR code ou export JSON — vous gardez le contrôle total.

4. Sécurité

  • Données stockées localement dans votre navigateur
  • Chiffrement via compression LZ-String pour la synchro
  • Effacement possible à tout moment depuis Paramètres
  • Fonctionne hors-ligne

5. Vos droits

  • Accéder à vos données
  • Exporter vos données (JSON)
  • Supprimer vos données
  • Utiliser l'application sans Internet

6. Cookies

BudgetCAD n'utilise aucun cookie ni outil d'analyse.

BudgetCAD est un gestionnaire de finances personnelles fonctionnant dans votre navigateur, fourni "tel quel" sans garantie.

3. Limitation de responsabilité
